### GoDone Project - A Task Management App

**Description**: GoDone is a React-powered frontend application designed for managing tasks efficiently with user-friendly interfaces.

### Features
- **Dashboard**: Provides a high-level overview of tasks.
- **Calendar**: Visualizes tasks and their deadlines on a calendar view.
- **Task Management**: Users can add, update, delete, and prioritize tasks.
- **User Profile Management**: Profile settings with image upload capabilities.
- **Responsive Design**: Optimized for desktop and mobile experiences.

### Tech Stack
- **React.js**: Frontend library.
- **Tailwind CSS**: For modern UI styling.
- **React Icons**: Icon libraries used for functional and aesthetic enhancements.

### How to Run
- Clone repo: `git clone https://github.com/FigoMandala/frontend-Go-Done.git`
- Navigate to the project directory: `cd frontend-Go-Done`
- Install dependencies: `npm install`
- Start development server: `npm start`

### Folder Structure
- **src/components/**: Contains reusable React components like Account, Dashboard, Sidebar, and Calendar.
- **src/main.jsx**: Entry point, includes routes and layouts.
- **LICENSE**: Details regarding project licensing.
